HTML

메인화면으로 돌아가기

  • HTML(Hyper Text Markup Language)이란 웹의 하이퍼 텍스트/미디어 문서를 작성하고 표현하는데 사용되는 표준 언어입니다. 또한 마크업 언어로 흔히 말하는 태그 (Tag)를 이용하여 작성하게 됩니다.

  • 표시할 정보가 있을 경우, 그 정보의 성격이나 기능을 태그라는 형식으로 두르는 것을 마크업이라고 한다. 이런 마크업 언어(Markup Language)로는 대표적인것이 HTML, XML이 있습니다.

  • 한가지 더 알고 가면 좋은 것은 HTTP(Hyper Text Transfer Protocol)이다.

  • HTML 문서를 송수신하기 위해 사용되는 통신규약이다.

  • 웹페이지를 방문하기 위해 적는 주소 첫 부분에 http 이라고 입력하는 것은 이 통신규약을 사용하기 때문이다.

  • 웹 페이지를 볼 수 있게 해주는 클라이언트 프로그램이 웹 브라우저이다.
